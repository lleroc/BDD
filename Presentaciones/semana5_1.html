Instrucción SELECT en SQL

La instrucción SELECT en SQL se utiliza para consultar y recuperar datos de una base de datos. Es una de las instrucciones más fundamentales y ampliamente utilizadas en SQL.

Sintaxis Básica

SELECT columna1, columna2, ... FROM nombre_tabla WHERE condición;

Esta sintaxis permite seleccionar datos específicos de una tabla, aplicando condiciones para filtrar los resultados.

Funciones Comunes Utilizadas con SELECT

SQL ofrece una variedad de funciones que se pueden utilizar junto con la instrucción SELECT para realizar operaciones en los datos. A continuación, se presentan algunas de las funciones más comunes:

1. Funciones de Agregación

  • COUNT(): Cuenta el número de filas que cumplen una condición específica.
  • SUM(): Calcula la suma total de un campo numérico.
  • AVG(): Calcula el promedio de un campo numérico.
  • MAX(): Devuelve el valor máximo de un campo.
  • MIN(): Devuelve el valor mínimo de un campo.

Ejemplo:

SELECT COUNT(*) AS total_empleados, AVG(salario) AS salario_promedio, MAX(salario) AS salario_maximo, MIN(salario) AS salario_minimo FROM empleados;

Compatible con: SQL Server, MySQL, PostgreSQL, Oracle.

2. Funciones de Cadena de Texto

  • UPPER(): Convierte una cadena de texto a mayúsculas.
  • LOWER(): Convierte una cadena de texto a minúsculas.
  • CONCAT(): Combina dos o más cadenas de texto (diferente en SQL Server, donde se utiliza +).
  • SUBSTRING(): Extrae una subcadena de una cadena de texto (se llama SUBSTR() en Oracle).
  • TRIM(): Elimina los espacios en blanco al inicio y al final de una cadena de texto.

Ejemplo:

SELECT CONCAT(UPPER(apellido), ', ', nombre) AS nombre_completo FROM empleados;

Compatible con: MySQL, PostgreSQL, Oracle. En SQL Server, se utiliza + en lugar de CONCAT().

3. Funciones de Fecha y Hora

  • NOW(): Devuelve la fecha y hora actuales (en SQL Server es GETDATE()).
  • DATE(): Extrae la parte de la fecha de un valor de fecha y hora.
  • YEAR(), MONTH(), DAY(): Extraen el año, mes y día de una fecha, respectivamente.
  • DATEDIFF(): Calcula la diferencia en días entre dos fechas (en Oracle se usa TO_DATE y resta).

Ejemplo:

SELECT nombre, fecha_contratacion, DATEDIFF(NOW(), fecha_contratacion) AS dias_en_empresa FROM empleados;

Compatible con: MySQL, PostgreSQL, SQL Server, Oracle (con ligeras diferencias).

4. Funciones Matemáticas

  • ROUND(): Redondea un número a un número específico de decimales.
  • CEIL(): Redondea un número al siguiente entero más alto (en SQL Server se utiliza CEILING()).
  • FLOOR(): Redondea un número al siguiente entero más bajo.
  • ABS(): Devuelve el valor absoluto de un número.

Ejemplo:

SELECT nombre, salario, ROUND(salario * 0.10, 2) AS bono FROM empleados;

Compatible con: SQL Server, MySQL, PostgreSQL, Oracle (redondeo similar).

Diferencias entre Bases de Datos

  • SQL Server: Usa GETDATE() para la fecha actual y + para concatenación.
  • MySQL: Soporta NOW(), CONCAT() y funciones directas de manejo de cadenas.
  • PostgreSQL: Ofrece gran flexibilidad con AGE() para cálculos de diferencia de fechas.
  • Oracle: Usa TO_DATE y SUBSTR(), y tiene su propio formato de concatenación.
