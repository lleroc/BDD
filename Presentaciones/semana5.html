Consultas en Bases de Datos

En esta presentación se describen las consultas más comunes que puedes realizar en una base de datos, con ejemplos prácticos y una explicación clara de su propósito y uso.

1. Consultas de Selección (SELECT)

Permiten recuperar datos de una o más tablas. Son las consultas más utilizadas en bases de datos y sirven para analizar y visualizar la información almacenada.

SELECT * FROM empleados; -- Selecciona todos los registros de la tabla empleados SELECT nombre, salario FROM empleados WHERE salario > 5000; -- Selecciona empleados con salario mayor a 5000

Ejemplo práctico: Si deseas ver todos los empleados que ganan más de un salario específico, puedes usar SELECT con una cláusula WHERE para filtrar los resultados.

2. Consultas de Inserción (INSERT)

Se utilizan para agregar nuevos registros a una tabla. Estas consultas son esenciales para registrar nueva información, como agregar un nuevo cliente o producto.

INSERT INTO empleados (nombre, edad, salario) VALUES ('Juan Pérez', 30, 4500); -- Inserta un nuevo empleado

Ejemplo práctico: Cuando contratas un nuevo empleado, puedes registrar sus datos en la tabla usando una consulta INSERT.

3. Consultas de Actualización (UPDATE)

Sirven para modificar los datos existentes en una tabla. Esto es útil cuando necesitas corregir o actualizar información previamente registrada.

UPDATE empleados SET salario = 5000 WHERE nombre = 'Juan Pérez'; -- Actualiza el salario de Juan Pérez

Ejemplo práctico: Si decides aumentar el salario de un empleado, puedes hacerlo utilizando una consulta UPDATE.

4. Consultas de Eliminación (DELETE)

Se utilizan para eliminar registros de una tabla. Esto es útil para borrar información que ya no es necesaria o que se registró por error.

DELETE FROM empleados WHERE nombre = 'Juan Pérez'; -- Elimina a Juan Pérez de la tabla

Ejemplo práctico: Si un empleado deja de trabajar en la empresa, puedes eliminar sus datos utilizando una consulta DELETE.

5. Consultas de Agregación

Permiten realizar operaciones matemáticas o estadísticas sobre los datos, como contar registros o calcular promedios.

SELECT COUNT(*) AS total_empleados FROM empleados; -- Cuenta el número de empleados SELECT AVG(salario) AS salario_promedio FROM empleados; -- Calcula el salario promedio

Ejemplo práctico: Para generar un informe del número total de empleados o el salario promedio en una empresa, puedes usar consultas de agregación.

6. Consultas de Joins

Se utilizan para combinar datos de dos o más tablas relacionadas. Son esenciales para trabajar con bases de datos relacionales.

SELECT empleados.nombre, departamentos.nombre AS departamento FROM empleados INNER JOIN departamentos ON empleados.id_departamento = departamentos.id; -- Une empleados y departamentos

Ejemplo práctico: Si deseas ver el nombre de cada empleado junto con el departamento al que pertenece, puedes usar un INNER JOIN.

7. Consultas de Creación de Tablas (CREATE TABLE)

Permiten definir la estructura de una nueva tabla, incluyendo sus columnas y tipos de datos.

CREATE TABLE empleados ( id INT AUTO_INCREMENT PRIMARY KEY, nombre VARCHAR(100), edad INT, salario DECIMAL(10, 2) ); -- Crea la tabla empleados

Ejemplo práctico: Cuando inicias un nuevo proyecto de base de datos, utilizas CREATE TABLE para definir las tablas necesarias.

8. Consultas de Eliminación de Tablas (DROP TABLE)

Eliminan por completo una tabla de la base de datos, incluyendo todos sus datos.

DROP TABLE empleados; -- Elimina la tabla empleados

Ejemplo práctico: Si ya no necesitas una tabla en tu base de datos, puedes eliminarla con DROP TABLE.
